# Right Ads Lead Management & Business Analytics Project

## Project Overview

This project focuses on analyzing a CRM lead dataset containing 17,278 business records. The objective was to clean, organize, analyze, and transform raw lead data into actionable business insights.

## Objectives

* Improve Data Quality
* Identify Duplicate Records
* Analyze Call Status Trends
* Generate Business Insights
* Build Executive Dashboard

## Dataset Overview

The dataset contains:

* Business Names
* Contact Details
* Phone Numbers
* Addresses
* Call Status Records
* Follow-Up Notes
* Additional Contact Information

## Data Cleaning Activities

* Duplicate Record Identification
* Missing Value Analysis
* Invalid Phone Number Detection
* Status Standardization
* Data Quality Assessment

## Exploratory Data Analysis (EDA)

* Lead Distribution Analysis
* Call Status Analysis
* Follow-Up Analysis
* Data Quality Analysis
* Lead Segmentation

## Key Findings

* Inconsistent Call Status Values
* Mixed Status Categories
* Blank Phone Numbers Found
* Multiple Status Spellings
* Follow-Up Notes Mixed with Status
* Data Standardization Required

## Recommendations

* Standardize Status Values
* Remove Invalid Numbers
* Categorize Follow-Up Notes
* Remove Duplicate Records
* Apply Validation Rules

* Cleaned Dataset

* Project Findings
* Status Mapping
* Executive Summary
* Recommendations
* Dashboard
