# Right Ads Lead Data Analysis Project

## Project Overview
This project focuses on analyzing lead data from Right Ads and identifying data quality issues affecting lead management and reporting.

## Dataset Information
- Total Records: 17,278
- Data Source: Right Ads Lead Dataset

## Key Findings
- Inconsistent Call Status Values
- Mixed Status Categories
- Blank Phone Numbers Found
- Multiple Status Spellings
- Follow-Up Notes Mixed With Status
- Data Standardization Required

## Recommendations
- Standardize Status Values
- Remove Invalid Numbers
- Categorize Follow-Up Notes
- Remove Duplicate Records
- Apply Validation Rules

- EDA Findings
